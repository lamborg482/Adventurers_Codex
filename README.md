# Интерактивный лист персонажа для D&D 5e

Веб‑приложение для удобного ведения персонажей в Dungeons & Dragons.  
Спроектировано для игроков и Мастеров подземелий: позволяет управлять характеристиками, инвентарём, заклинаниями и боевыми параметрами в реальном времени.

## Возможности

- **Полный учёт персонажа**  
  Характеристики, навыки, спасброски, пассивная внимательность — всё автоматически рассчитывается.
- **Инвентарь и экипировка**  
  Добавление предметов, отслеживание веса, стоимости, категорий снаряжения.
- **Книга заклинаний**  
  Список заклинаний с фильтрацией по уровню, школе, компонентам; учёт ячеек.
- **Боевой модуль**  
  Хиты, класс брони, инициатива, временные эффекты и состояния.
- **Режим Мастера**  
  Сводка по всем персонажам партии, быстрое редактирование, мониторинг ключевых параметров.
- **Адаптивный интерфейс**  
  Комфортная работа на компьютерах, планшетах и смартфонах.

## Стек технологий

### Frontend
- **Vue.js 3** — реактивный интерфейс
- **Vue Router** — навигация
- **CSS3 / JavaScript** — стилизация и логика
- **Fetch API / Axios** — взаимодействие с бэкендом

### Backend
- **FastAPI** — высокопроизводительный веб‑фреймворк
- **SQLAlchemy** — ORM для работы с базой данных
- **Pydantic** — валидация данных
- **PostgreSQL** — СУБД
