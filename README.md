# Интерактивный лист персонажа для D&D 5e

Веб‑приложение для ведения персонажей в Dungeons & Dragons пятой редакции.  
Проект создан для упрощения игрового процесса как со стороны игроков, так и со стороны Мастера подземелий (DM).  
Вся информация о персонажах хранится локально или синхронизируется через выбранное вами хранилище (поддержка планируется).

## Возможности

- **Характеристики и навыки**  
  Автоматический расчёт модификаторов, спасбросков, пассивной внимательности и других параметров.
- **Инвентарь**  
  Учёт предметов, их веса, стоимости, категорий экипировки.
- **Книга заклинаний**  
  Список известных и подготовленных заклинаний, ячейки заклинаний, фильтрация по школам и уровням.
- **Боевой модуль**  
  Отслеживание хитов, класса брони, инициативы, временных эффектов.
- **Режим Мастера**  
  Сводка по всем персонажам партии, быстрый просмотр основных параметров, возможность редактирования.
- **Адаптивный дизайн**  
  Удобная работа на компьютерах, планшетах и телефонах.
